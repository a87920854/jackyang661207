活動說明 Event Description

妳是擁有自信成熟穩重的姊姊型女孩，還是個子小、頭髮短， 笑起來天真爛漫的精豆型女孩? 亦或是一個清爽的(偽)素顏型女生。 即日起上傳妳的有型照片，由網友票選出前三名，將獲得精美禮物/禮金一份。

活動流程 Schedule

  1. 加入春天會館-全方位健康交友平台(單身俱樂部)粉絲團。
  2. 於活動網頁上傳妳最有型、最自信、最美的照片。
  3. 於活動期間【即日起~2017/09/05(二)24:00止】廣邀網友來按讚及分享。
  4. 由網友票選出前三名高票，將贈送精美好禮/禮金一份。

活動時間 Activity Time

線上投票時間：即日起~2017/9/5止

參加資格 Qualifications

  1. 參加璀璨之星資格：女性限71~86年次。
  2. 投票資格：所有網友皆可參加。

活動獎項 Awards

  1. 票選第一名：獎金NT10,000
  2. 票選第二名：(待確認)
  3. 票選第三名：(待確認)

獲獎公告時間 Announcement

  1. 2017年09月12日(二) 12:00以前

得獎公布 Who's the Winner

  1. 活動結束後，得獎名單會公佈在春天會館活動平台公告。
  2. 名單公佈後，將以電話通知得獎者，中獎者須在7日內(2017年09月12日 中午12:00)至春天會館-全方位健康交友平台(單身俱樂部)粉絲團私訊填寫收件資料，逾期視同放棄領獎機會。
